startUpgrade

expect abstract fun startUpgrade(): SDKRequest<Unit>

Initiates the upgrade process.

This method starts the upgrade process using the latest available upgrade information. The actual upgrade process runs asynchronously and progress can be monitored through upgrade events.

Example:

upgradeService.startUpgrade()
    .proceed(object : SDKRequestCallback<Unit> {
        override fun onSuccess(unit: Unit?) {
            println("Upgrade started")
        }
    })
