IAppUpgradeInfo

interface IAppUpgradeInfo

Interface defining application upgrade information structure.

This interface encapsulates all metadata and configuration details for an application upgrade package, including version information, package details, and compatibility requirements.

Example usage:

upgradeService.checkUpgrade()
    .proceed(object : SDKRequestCallback<List<IAppUpgradeInfo>> {
        override fun onSuccess(upgrades: List<IAppUpgradeInfo>?) {
            upgrades?.forEach { upgrade ->
                // Check upgrade type
                if (upgrade.upgradeType == MANDATORY_UPGRADE) {
                    println("Mandatory upgrade required:")
                    println("Version: ${upgrade.versionName}")
                    println("Size: ${upgrade.size} bytes")
                    println("Message: ${upgrade.upgradeMsg}")
                }
            }
        }
    })
