UpgradeEvent

class UpgradeEvent(event: Enum<UpgradeEventType>) : SDKEvent<IUpgradeService, UpgradeEventType>

Class representing upgrade events in the system.

This class extends SDKEvent to handle upgrade-related events from IUpgradeService. It provides type-safe event handling for upgrade operations, particularly Docker container upgrades.

Example usage:

upgradeService.registerUpgradeEvent { event ->
    // Type-safe event handling
    when (event.type) {
        UpgradeEventType.UPGRADE_DOCKER_EVENT -> {
            // Handle Docker upgrade event
            println("Docker upgrade event received")
            println("Event source: ${event.from}")
            println("Event type: ${event.type}")
        }
    }
}
