OrderStatus

enum OrderStatus : Enum<OrderStatus>

Enumeration representing all possible states of an order throughout its lifecycle.

This enum defines the complete set of states that an order can be in, including initial payment processing, successful completion, failures, and post-purchase operations like chargebacks and refunds.

Example usage:

// Query orders with specific statuses
billService.queryOrderList(listOf(
    OrderStatus.PAYMENT_PENDING,
    OrderStatus.PAYMENT_SUCCEEDED
)).proceed()
