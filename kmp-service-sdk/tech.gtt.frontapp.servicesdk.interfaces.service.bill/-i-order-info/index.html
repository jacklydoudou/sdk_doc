IOrderInfo

interface IOrderInfo

Interface representing detailed order information.

This interface defines the complete structure of an order, including order details, payment information, and status tracking. It provides comprehensive information about an order throughout its lifecycle.

Example usage:

billService.queryOrderList(listOf(OrderStatus.COMPLETED)).proceed()
