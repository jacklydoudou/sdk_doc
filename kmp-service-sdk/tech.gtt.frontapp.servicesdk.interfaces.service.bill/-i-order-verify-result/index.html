IOrderVerifyResult

interface IOrderVerifyResult

Interface representing the result of an order verification operation.

This interface defines the structure for results returned when verifying an order's status and purchase details. It provides essential information about the order's validity and current state.

Example usage:

billService.verifyOrder(orderId).proceed()
