IBillService

expect interface IBillService : IService, ITryWatch

Interface for managing billing operations including subscriptions, orders, and payment processing.

This service provides comprehensive billing functionality including:

  • Subscription management

  • Order processing

  • Payment handling

  • Google Play Billing integration

Example usage:

// Initialize billing client
billService.startBillClient(context)

// Query subscription packages
billService.querySubscriptionList().execute { result ->
    result.onSuccess { subscriptions ->
        // Handle subscription list
    }.onFailure { error ->
        // Handle error
    }
}
