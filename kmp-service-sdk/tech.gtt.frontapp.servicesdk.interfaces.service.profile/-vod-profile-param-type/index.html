VodProfileParamType

enum VodProfileParamType : Enum<VodProfileParamType>

Enumeration of VOD (Video On Demand) profile parameter types.

This enum defines the different types of VOD profile parameters used in the system, categorizing operations for viewing history and favorites management.

Example usage:

// Add to history
val params = VodProfileParam.companion.create()
vodProfile.addVod(AddVodProfileParam.companion.history(params))
    .proceed(object : SDKRequestCallback<Unit> {
        override fun onSuccess(unit: Unit?) {
            println("Added to history")
        }
    })

// Query favorites
vodProfile.getVodList(
    QueryVodProfileParam.companion.favorite(),
    forceSync = true,
    pageNo = 1,
    pageSize = 20
).proceed(object : SDKRequestCallback<List<IProfileVodInfo>> {
    override fun onSuccess(content: List<IProfileVodInfo>?) {
        println("Found ${content?.size} favorites")
    }
})
