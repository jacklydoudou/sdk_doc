isVodFavored

expect abstract fun isVodFavored(addedVodParam: HasFavoredVodProfileParam): SDKRequest<List<IVodHasFavorInfo>>

Checks if VOD entries are in favorites.

This method checks the favorite status of specified VOD entries. It's useful for UI state management (e.g., favorite button states) and batch status checks. The operation is asynchronous and returns a SDKRequest that must be executed using proceed() with a callback.

Example:

vodProfile.isVodFavored(HasFavoredVodProfileParam.companion.favorite(listOf("123", "456")))
    .proceed(object : SDKRequestCallback<List<IVodHasFavorInfo>> {
        override fun onSuccess(statuses: List<IVodHasFavorInfo>?) {
            statuses?.forEach { status ->
                println("ID: ${status.id}, Favorited: ${status.isFavored}")
            }
        }
        override fun onFail(exception: SDKException) {
            println("Failed to check favorite status")
        }
    })
