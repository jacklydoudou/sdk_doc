IVodProfile

expect interface IVodProfile

Interface for managing VOD (Video On Demand) profile configurations.

This interface provides functionality for managing user interactions with VOD content, including viewing history, favorites, and content preferences. All operations are asynchronous and return SDKRequest objects that must be executed using proceed().

Example usage:

// Get VOD profile from service
val vodProfile = serviceSDK.getProfileService().getVodProfile()

// Create content parameters
val params = VodProfileParam.Companion.create();

// Add to favorites
vodProfile.addVod(AddVodProfileParam.favorite(params))
    .proceed(object : SDKRequestCallback<Unit> {
        override fun onSuccess(unit: Unit?) {
            println("Content added to favorites")
        }
        override fun onFail(exception: SDKException) {
            println("Failed to add content")
        }
    })
