expect abstract fun deleteVod(deleteVodParam: DeleteVodProfileParam): SDKRequest<Unit>

Removes a VOD entry from history or favorites.

This method removes a VOD entry from either the viewing history or favorites list. The operation is asynchronous and returns a SDKRequest that must be executed using proceed() with a callback.

Example:

vodProfile.deleteVod(DeleteVodProfileParam("123", VodProfileParamType.FAVORITE))
    .proceed(object : SDKRequestCallback<Unit> {
        override fun onSuccess(unit: Unit?) {
            println("Content removed from favorites")
        }
        override fun onFail(exception: SDKException) {
            println("Failed to remove content")
        }
    })
