QueryVodProfileParam

expect class QueryVodProfileParam

Class for querying VOD (Video On Demand) profile parameters 点播配置参数查询类

This class provides functionality to query VOD history and favorites. Use the companion object factory methods history and favorite to create instances. The query results can be retrieved using IVodProfile.getVodList.
