AddVodProfileParam

expect class AddVodProfileParam

Class for adding VOD (Video On Demand) profile entries 添加点播配置条目的类

This class provides functionality to add VOD history and favorites entries. Use the companion object factory methods history and favorite to create instances.

该类提供添加点播历史和收藏条目的功能。 使用伴生对象工厂方法history和favorite创建实例。

Example usage:

// Create VOD profile parameters
val vodParams = VodProfileParam(
    vodId = "123",
    title = "Movie Title",
    duration = 7200 // 2 hours in seconds
)

// Add to history
val historyParam = AddVodProfileParam.history(vodParams)
historyParam?.let { param ->
    vodProfile.addVod(param).execute()
}

// Add to favorites
val favoriteParam = AddVodProfileParam.favorite(vodParams)
favoriteParam?.let { param ->
    vodProfile.addVod(param).execute()
}
