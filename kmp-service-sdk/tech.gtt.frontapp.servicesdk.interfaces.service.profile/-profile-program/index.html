ProfileProgram

interface ProfileProgram

Interface representing program information in profile operations.

This interface defines the structure for program data returned by profile operations such as querying program subscriptions and handling subscription events. It provides access to program metadata including identifiers, timestamps, and detailed program information.

Example usage:

liveProfile.queryProgramSubscribeList()
    .proceed(object : SDKRequestCallback<List<ProfileProgram>> {
        override fun onSuccess(programs: List<ProfileProgram>?) {
            programs?.forEach { program ->
                println("Channel: ${program.logicalChannelId}")
                println("Program: ${program.programId}")
                println("Last Updated: ${program.updateTime}")
                println("EPG ID: ${program.epgId}")
            }
        }
    })

// Handle subscription events
liveProfile.registerProgramSubscribeEvent { event ->
    when (event.type) {
        ProfileEventType.SUBSCRIBE_UPDATE -> {
            val program = event.data as ProfileProgram
            println("Subscription updated for:")
            println("Channel: ${program.logicalChannelId}")
            println("Program: ${program.programId}")
        }
    }
}
