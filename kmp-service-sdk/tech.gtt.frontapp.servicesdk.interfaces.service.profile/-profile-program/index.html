interface ProfileProgram

Interface providing program information for user profiles 提供用户配置的节目单信息接口

This interface defines the structure for program information that is associated with user profiles. It combines channel information with EPG (Electronic Program Guide) data to provide a complete view of a program in the context of user profiles.

该接口定义了与用户配置相关联的节目信息的结构。 它将频道信息与电子节目指南(EPG)数据结合在一起，以提供用户配置环境下节目的完整视图。
