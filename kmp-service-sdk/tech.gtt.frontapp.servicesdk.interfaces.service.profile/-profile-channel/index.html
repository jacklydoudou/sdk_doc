ProfileChannel

interface ProfileChannel

Interface representing channel information in profile operations.

This interface defines the structure for channel data returned by profile operations such as querying channel history or favorites. It provides access to channel metadata including unique identifier, update timestamp, and detailed channel information.

Example usage:

liveProfile.queryChannel(QueryChannelParam.companion.favorite())
    .proceed(object : SDKRequestCallback<List<ProfileChannel>> {
        override fun onSuccess(channels: List<ProfileChannel>?) {
            channels?.forEach { channel ->
                println("Channel ID: ${channel.logicalChannelId}")
                println("Last Updated: ${channel.updateTime}")
                println("Channel Data: ${channel.channelData}")
            }
        }
    })
