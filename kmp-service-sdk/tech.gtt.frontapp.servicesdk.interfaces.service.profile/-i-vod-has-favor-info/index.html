interface IVodHasFavorInfo

Interface for VOD (Video On Demand) favorite status information 点播收藏状态信息接口

This interface provides information about whether a VOD item is in favorites. Used in conjunction with IVodProfile.isVodFavored to check favorite status of VOD content.
