IVodHasFavorInfo

interface IVodHasFavorInfo

Interface representing VOD (Video On Demand) favorite status information.

This interface defines the structure for VOD favorite status data returned by profile operations checking whether content is in the user's favorites. It provides access to both the content ID and its favorite status.

Example usage:

vodProfile.isVodFavored(
    HasFavoredVodProfileParam.companion.favorite(listOf("123", "456"))
).proceed(object : SDKRequestCallback<List<IVodHasFavorInfo>> {
    override fun onSuccess(statuses: List<IVodHasFavorInfo>?) {
        statuses?.forEach { status ->
            println("VOD ${status.getId()} is favorite: ${status.getIsFavored()}")
        }
    }
})
