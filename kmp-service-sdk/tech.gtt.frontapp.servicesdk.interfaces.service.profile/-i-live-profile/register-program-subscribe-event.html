registerProgramSubscribeEvent

expect abstract fun registerProgramSubscribeEvent(action: ICallback<ProfileEvent>): SDKDispose

Registers for program subscription events 注册节目订阅事件

This method sets up a callback for receiving program subscription updates 此方法设置用于接收节目订阅更新的回调

Example:

val dispose = liveProfile.registerProgramSubscribeEvent { event ->
    when (event.type) {
        ProfileEventType.SUBSCRIBE -> {
            val program = event.data as ProfileProgram
            println("New subscription: ${program.name}")
        }
        ProfileEventType.UNSUBSCRIBE -> {
            val program = event.data as ProfileProgram
            println("Unsubscribed: ${program.name}")
        }
    }
}

// Clean up when done
dispose.dispose()
