ILiveProfile

expect interface ILiveProfile

Interface for managing live streaming profile configurations and interactions.

This interface provides functionality for managing user interactions with live streaming content, including channel management (history and favorites) and program subscriptions.

Example usage:

// Get live profile from service
val liveProfile = serviceSDK.getProfileService().getLiveProfile()

// Add channel to history
liveProfile.addChannel(AddChannelParam.history("18782"))
    .proceed(object : SDKRequestCallback<Unit> {
        override fun onSuccess(unit: Unit?) {
            println("Channel added to history")
        }
        override fun onFail(exception: SDKException) {
            println("Failed to add channel")
        }
    })

// Query favorite channels
liveProfile.queryChannel(QueryChannelParam.favorite())
    .proceed(object : SDKRequestCallback<List<ProfileChannel>> {
        override fun onSuccess(channels: List<ProfileChannel>?) {
            channels?.forEach { channel ->
                println("Channel ID: ${channel.logicalChannelId}")
            }
        }
    })
