Parameters class for adding channel profile operations 添加频道配置操作的参数类

This class provides functionality to create parameters for adding channels to either viewing history or favorites list. 该类提供创建参数的功能，用于将频道添加到观看历史或收藏列表。

Main features: 主要功能：

  1. Create history parameters for tracking viewed channels 创建用于跟踪已观看频道的历史参数

  2. Create favorite parameters for bookmarking channels 创建用于收藏频道的参数
