ProfileEvent

class ProfileEvent(event: Enum<ProfileEventType>, data: ProfileProgram) : SDKEvent<IProfileService, ProfileEventType>

Data class representing profile-related events.

This class encapsulates event data for profile operations, particularly program subscription updates. It contains the event type and associated data.

Example usage:

liveProfile.registerProgramSubscribeEvent { event ->
    when (event.type) {
        ProfileEventType.SUBSCRIBE_UPDATE -> {
            val program = event.data as ProfileProgram
            println("Program subscription updated:")
            println("Channel: ${program.logicalChannelId}")
            println("Program: ${program.programId}")
        }
    }
}
