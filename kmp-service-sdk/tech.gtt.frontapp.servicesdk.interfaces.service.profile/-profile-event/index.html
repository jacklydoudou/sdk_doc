ProfileEvent

class ProfileEvent(event: Enum<ProfileEventType>, data: ProfileProgram) : SDKEvent<IProfileService, ProfileEventType>

Profile event class that represents events related to user profiles 用户配置事件类，表示与用户配置相关的事件

This class extends SDKEvent to handle profile-specific events in the service SDK. It is used to notify subscribers about changes in user profile data, particularly for VOD and subscription-related updates.

该类继承自SDKEvent以处理服务SDK中的配置相关事件。 用于通知订阅者用户配置数据的变化，特别是点播和订阅相关的更新。
