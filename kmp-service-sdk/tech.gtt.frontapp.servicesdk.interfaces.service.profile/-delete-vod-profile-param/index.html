DeleteVodProfileParam

expect class DeleteVodProfileParam

Parameters class for deleting VOD (Video On Demand) profile entries.

This class provides functionality to create parameters for removing VOD entries from either viewing history or favorites list. Use the companion object's factory methods to create instances.

Example usage:

// Delete from history
vodProfile.deleteVod(DeleteVodProfileParam.companion.history("123"))
    .proceed(object : SDKRequestCallback<Unit> {
        override fun onSuccess(unit: Unit?) {
            println("Removed from history")
        }
        override fun onFail(exception: SDKException) {
            println("Failed to remove from history")
        }
    })

// Delete from favorites
vodProfile.deleteVod(DeleteVodProfileParam.companion.favorite("123"))
    .proceed(object : SDKRequestCallback<Unit> {
        override fun onSuccess(unit: Unit?) {
            println("Removed from favorites")
        }
        override fun onFail(exception: SDKException) {
            println("Failed to remove from favorites")
        }
    })
