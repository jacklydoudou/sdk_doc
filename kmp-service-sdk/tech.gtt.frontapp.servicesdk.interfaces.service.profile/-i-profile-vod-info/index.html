IProfileVodInfo

interface IProfileVodInfo

Interface representing VOD (Video On Demand) content information in profile operations.

This interface defines the structure for VOD content data returned by profile operations such as querying viewing history or favorites. It provides access to essential content metadata like title, duration, and playback state.

Example usage:

vodProfile.getVodList(
    queryVodParam = QueryVodProfileParam.companion.history(),
    forceSync = true,
    pageNo = 1,
    pageSize = 20
).proceed(object : SDKRequestCallback<List<IProfileVodInfo>> {
    override fun onSuccess(content: List<IProfileVodInfo>?) {
        content?.forEach { vod ->
            println("Title: ${vod.getTitle()}")
            println("Duration: ${vod.getDuration()}")
            println("Watched: ${vod.getPlayedTime()}")
        }
    }
})
