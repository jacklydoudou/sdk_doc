IProfileVodInfo

interface IProfileVodInfo

Interface for VOD (Video On Demand) profile information 点播配置信息接口

This interface provides comprehensive information about a VOD item, including basic metadata, series information, and playback history. Used in conjunction with IVodProfile methods to manage VOD content.

该接口提供点播项目的完整信息，包括基本元数据、系列信息和播放历史。 与IVodProfile方法一起使用来管理点播内容。
