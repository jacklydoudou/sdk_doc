VodProfileParam

expect class VodProfileParam

Class representing VOD (Video On Demand) profile parameters.

This class encapsulates metadata and state information for VOD content, including basic information and user-specific data. Use the companion object's create() method to instantiate new parameters.

Example usage:

// Create parameters using companion method
val params = VodProfileParam.companion.create()

// Use parameters with VOD operations
vodProfile.addVod(AddVodProfileParam.companion.favorite(params))
    .proceed(object : SDKRequestCallback<Unit> {
        override fun onSuccess(unit: Unit?) {
            println("Content added to favorites")
        }
    })
