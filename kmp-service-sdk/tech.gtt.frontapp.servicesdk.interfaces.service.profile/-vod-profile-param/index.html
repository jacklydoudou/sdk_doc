Class representing VOD (Video On Demand) profile parameters 点播配置参数类

This class encapsulates all metadata and state information for a VOD content item, including basic information like title and duration, as well as user-specific data like playback position and watch status.

该类封装了点播内容项的所有元数据和状态信息， 包括标题和时长等基本信息，以及播放位置和观看状态等用户特定数据。
