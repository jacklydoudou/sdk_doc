getVodProfile

expect abstract fun getVodProfile(): IVodProfile

Retrieves the VOD profile service for managing video-on-demand preferences.

This service handles all operations related to VOD content profiles, including:

  • Content favorites management

  • Viewing history tracking

  • Playback position management

Example:

val vodProfile = profileService.getVodProfile()

// Add content to favorites
val params = VodProfileParam().apply {
    metaId = "123"
    title = "Movie Title"
    duration = 7200000 // 2 hours in milliseconds
}
vodProfile.addVod(AddVodProfileParam.favorite(params))
    .proceed(object : SDKRequestCallback<Unit> {
        override fun onSuccess(unit: Unit?) {
            println("Content added to favorites")
        }
        override fun onFail(exception: SDKException) {
            println("Failed to add content")
        }
    })
