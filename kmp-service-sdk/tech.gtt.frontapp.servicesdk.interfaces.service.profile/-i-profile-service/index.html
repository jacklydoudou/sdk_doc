IProfileService

expect interface IProfileService : IService

Core interface for managing user profile preferences and content interactions.

This service provides functionality for managing user preferences and interactions with both live and VOD content. It serves as the main entry point for all profile-related operations in the SDK.

Example usage:

// Get profile service instance
val profileService = serviceSDK.getProfileService()

// Access live profile functionality
val liveProfile = profileService.getLiveProfile()

// Add channel to history
liveProfile.addChannel(AddChannelParam.history("18782"))
    .proceed(object : SDKRequestCallback<Unit> {
        override fun onSuccess(unit: Unit?) {
            println("Channel added to history")
        }
        override fun onFail(exception: SDKException) {
            println("Failed to add channel")
        }
    })

// Query favorite channels
liveProfile.queryChannel(QueryChannelParam.favorite())
    .proceed(object : SDKRequestCallback<List<ProfileChannel>> {
        override fun onSuccess(channels: List<ProfileChannel>?) {
            channels?.forEach { channel ->
                println("Channel ID: ${channel.logicalChannelId}")
            }
        }
        override fun onFail(exception: SDKException) {
            println("Failed to query channels")
        }
    })
