HasFavoredVodProfileParam

expect class HasFavoredVodProfileParam

Parameters class for checking VOD (Video On Demand) favorite status.

This class provides functionality to create parameters for checking whether specified VOD entries are in the user's favorites list. Use the companion object's factory method to create instances.

Example usage:

// Check favorite status for multiple VOD entries
vodProfile.isVodFavored(
    HasFavoredVodProfileParam.companion.favorite(listOf("123", "456"))
).proceed(object : SDKRequestCallback<List<IVodHasFavorInfo>> {
    override fun onSuccess(statuses: List<IVodHasFavorInfo>?) {
        statuses?.forEach { status ->
            println("VOD ${status.id} is favorite: ${status.isFavored}")
        }
    }
    override fun onFail(exception: SDKException) {
        println("Failed to check favorite status")
    }
})
