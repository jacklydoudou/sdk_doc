deleteProgSubscribeParam

expect fun deleteProgSubscribeParam(logicalChannelId: String, programId: String): DeleteProgSubscribeParam?

Creates parameters for unsubscribing from a program.

Factory method to create parameters for removing a program subscription. The operation is performed using ILiveProfile.deleteProgramSubscribe.

Example:

val param = DeleteProgSubscribeParam.companion.deleteProgSubscribeParam(
    channelId = "18783",
    programId = "0"
)
liveProfile.deleteProgramSubscribe(listOf(param))
    .proceed(object : SDKRequestCallback<Unit> {
        override fun onSuccess(unit: Unit?) {
            println("Program unsubscribed")
        }
    })
