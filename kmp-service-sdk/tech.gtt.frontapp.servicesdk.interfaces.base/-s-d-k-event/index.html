SDKEvent

abstract class SDKEvent<T : IService, Type : Enum<Type>>(var from: KClass<T>, var event: Enum<Type>)

Abstract base class for SDK events.

This class represents events that can be emitted by various services in the SDK. It provides type-safe event handling with service-specific event types.
