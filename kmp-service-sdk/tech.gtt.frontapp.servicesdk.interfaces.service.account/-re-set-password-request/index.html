ReSetPasswordRequest

expect class ReSetPasswordRequest

Password reset request class

This class handles password reset requests for both phone and email accounts, requiring verification codes for security.

if you want to use it, you can use the one of the following methods:

  • phone: creates a phone account password reset request.

  • email: creates an email account password reset request.

Example Usage

val result = ReSetPasswordRequest.phone("phone account", "new password", "areaCode", "verification code")
