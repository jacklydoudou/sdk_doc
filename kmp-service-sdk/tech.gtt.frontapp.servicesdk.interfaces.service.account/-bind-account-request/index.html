BindAccountRequest

expect class BindAccountRequest

Account binding request class.

This class handles requests for binding phone numbers or email addresses to existing accounts. It supports binding both new and existing accounts.

Companion Methods

The following companion methods are available for creating instances of BindAccountRequest:

  • bindNewEmail: bind a new email account.

  • bindExistEmail: bind an existing email account.

  • bindNewPhone: bind a new phone account.

  • bindExistPhone: bind an existing phone account.

Example Usage

val bindAccountRequest = BindAccountRequest.bindNewEmail(
    account = "user@example.com",
    password = "password123",
    verificationCode = 123456
)
