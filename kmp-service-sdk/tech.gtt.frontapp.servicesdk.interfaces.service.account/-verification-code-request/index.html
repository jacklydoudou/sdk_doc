VerificationCodeRequest

expect class VerificationCodeRequest

Verification code request class

This class handles requests for verification codes used in account registration, password reset, and other authentication processes.

if you want to use it, you can use the one of the following methods:

  • phone: creates a phone verification code request.

  • email: creates an email verification code request.

Example Usage

val result = VerificationCodeRequest.phone("phone account", "areaCode", false)
